Affinity Diagram

Using my Project Management 2 problem statement, I brainstormed ideas, grouped them by theme, and labeled each cluster to surface patterns and next steps.

Root Causes

  • Unclear onboarding steps
  • Inconsistent handoff between teams
  • Outdated docs

Pain Points

  • Duplicate work across tools
  • Slow approvals
  • Hard to find owners

Opportunities

  • Single source of truth
  • Auto-generated checklists
  • Role-based templates

Quick Wins

  • Define owners per step
  • Archive stale docs
  • Publish a FAQ

Grouped ideas highlight patterns and guide solution directions.
